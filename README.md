 # Crypto Vault Website 🚀

A responsive static website deployed on AWS EC2 using Linux and Nginx.

## 🛠️ Technologies Used

* AWS EC2
* Amazon Linux
* Nginx
* HTML
* CSS
* JavaScript
* Git & GitHub

## 📌 Features

* Responsive Design
* Cloud Deployment
* Linux Server Hosting
* Nginx Configuration

## 🚀 Deployment Process

1. Launch EC2 Instance
2. Connect using SSH
3. Transfer files using SCP
4. Install and configure Nginx
5. Deploy website to server
